Everything from shape?

The only thing you know about a NumPy array a is its shape:

print(a.shape)  # (3, 2, 6, 1)

What else can you infer from this knowledge?

Do you know:

  • its number of dimensions?
  • its size?
  • the number of elements in the first dimension?

Sum the numerical values of the features that can be inferred from a.shape . If the feature can't be inferred, do not add anything to the score. The answer is the sum described above.

For example, let's say we can get the number of dimensions from a.shape, and it's 3; we can't infer the size from a.shape; and the number of elements in the first dimension is 5. Then, the answer would be \(3+5 = 8\).

If you are not sure about the answer, revise the information from the "Learning sizes" part in the theory. You can also consider the following snippet to obtain the answer: 
import numpy as np
n = np.zeros((3, 2, 6, 1))
